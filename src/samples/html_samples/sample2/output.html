Tags and texts are separated in this sample, which means a tag and a text can't be mixed in a line.

You can write a long sentence without line concatenation.

A normal line is interpreted as a text line automatically, but tags must be prefixed with ">\", which is a bit noisy.

An empty line translates to a <BR> tag.

These settings are written in a first line. See "(repository_root)/lang_spec.txt" about it.
